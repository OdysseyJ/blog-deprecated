# Amplitude - 모르는 단어 정리

- taxonomy : 분류
- segmentation : 분할
- Event segmentation : 사용자의 행동을 이벤트에 따라 세분화하는 것
- cohort : 통계상의 인자를 공유하는 집단
- Event Properties : 이벤트가 발생할 당시의 context
- funnel : 깔때기
- funnel analysis : 유저들이 들어온 시점부터 나가는 시점까지를 분석해 나가는 시점과
  이유를 밝히는 것 (이탈률)
- retention : 유지율
- referral : 추천
- revenue : 구매와 결제(매출)
- compoisition : 구성
- attribution : touchpoint와 conversion 을 수집하고 이들을 정해진 규칙에 따라 연결시키는
  것.
- touchpoint : 제품이나 소비자가 만나는 접점, 보통 앱/웹에서는 click과 impression
- conversion : 구매를 비롯하여 구매를 촉진시킬 수 있는 고객의 행동 (전환)
