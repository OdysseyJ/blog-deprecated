# boj-2609

```python
a, b = map(int, input().split(" "))

def gcd(a, b):
    if b==0:
        return a;
    return gcd(b, a%b)

def lcm(a, b):
    return int((a*b)/gcd(a, b))

print(gcd(a, b))
print(lcm(a, b))
```
